# Ability Re 续接文档

## 当前定位

前后端分离官网学习项目：

- 前端选择 SvelteKit / Svelte 5 / TypeScript，用于学习区别于 Vue 的编译型前端框架。
- 后端选择 Java 21 / Spring Boot 3，用于学习主流企业后端分层、REST API、参数校验和数据库访问。
- 数据库改为 MySQL，方便使用本地 Navicat 连接、查看和维护数据。
- Flyway 管理 schema 和初始化内容。

## 已完成结构

```text
frontend/    SvelteKit 官网页面
backend/     Spring Boot API 服务
docker-compose.yml MySQL 本地数据库，可选
```

## 已实现功能

- `GET /api/health`
- `GET /api/site/home`
- `POST /api/contact`
- `site_sections`、`site_features`、`contact_messages` 三张表
- 前端首页、能力卡片、联系表单
- 前端 API 不可用时显示兜底内容，启动后端后读取数据库内容

## MySQL / Navicat 信息

默认连接：

```text
Host: localhost
Port: 3306
Database: ability_re
User: ability_re
Password: ability_re_password
```

如果使用本机已有 MySQL，需要先创建数据库和用户；SQL 已写在 README。

## 验证状态

- 前端已通过：`pnpm lint && pnpm test && pnpm build`
- 后端未能在当前机器验证：当前环境没有 Java Runtime、Maven，Docker 也不可用；需要本机安装 Java 21 + Maven 后运行 `mvn test`。

## 后续建议

1. 安装或确认 Java 21、Maven。
2. 用 Navicat 创建 `ability_re` 数据库和 `ability_re` 用户，或使用 Docker MySQL。
3. 执行 `cd backend && mvn spring-boot:run`，让 Flyway 自动建表。
4. 前端执行 `cd frontend && pnpm dev`。
5. 下一阶段可以增加：后台管理、内容 CRUD、联系线索列表、登录权限。
